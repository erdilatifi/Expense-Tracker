# Backend (PHP + MySQL)

## Setup

1. Create the database + tables:

- Run `database/schema.sql` in your MySQL client.

2. Configure env:

- Copy `.env.example` to `.env` and update values.

3. Run the API:

- From `backend/` run:
  - `php -S localhost:8000`

Endpoints live under `http://localhost:8000/api/...`.
